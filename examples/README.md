# Examples

These are canonical example artifacts for PolyBash.

- `fighter/project.zxmodel.json` — example fighter project
- `stylepacks/zx_fighter_v1.stylepack.json` — starter style pack
- `commands/fighter_variant_commands.json` — example structured edit command list

These examples are intended to be:
- human-readable
- deterministic
- reused in tests
- kept aligned with the docs
